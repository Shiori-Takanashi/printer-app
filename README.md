# プロジェクト概要

このプロジェクトは、元々以下の2つのリポジトリに分かれていました。

- `printer-backend`（Pythonによるデータ加工処理,フレームワーク無し）
- `printer-frontend`（Next.jsによるフロントエンド表示）

現在は、これらを **Vite + React** に統一し、よりシンプルな構成へ移行しています。

---

# 目的

- 技術スタックをReactに統一して、わかりやすくする。
- Next.jsを、わざわざ使わない。
- 小規模プロジェクトを目指す。
- 無駄なPythonを削り、撤廃する。

---
